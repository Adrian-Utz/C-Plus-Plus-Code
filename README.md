# C-Plus-Plus-Code
C++ Coding Practice


## Files

main.cpp - Entrance point for the program.

binary2digit.cpp - Convert a binary number to its decimal counterpart.
E.G. 10101111 = 175. To activate call binaryToDecimal(int).

burgermaker.cpp - Make your own burger.

helloworld.cpp - Classic

keytranslator.cpp - Tells you the values of a pressed key.
E.G. "A" equals 97 in ASCII and 01100001 in binary.
